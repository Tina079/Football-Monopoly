# ⚽ 足球大富翁

一款融合大富翁与足球经理的本地多人棋盘游戏。买入俱乐部、签下球员、升级球场，以六维属性对战一决胜负。从国内联赛一路打到欧冠，率先集齐 100 kw 资金、3 座五级球场和一座欧冠奖杯的玩家获胜。支持 2–4 人，可混入 🤖 机器人补齐人数。

## 🎮 游戏规则

### 基础信息

| 项目 | 说明 |
|------|------|
| 支持人数 | 2–4 人（可混入 🤖 机器人） |
| 初始资金 | 每人 10 kw |
| 棋盘 | 40 格，11×11 外围一圈顺时针行进 |
| 破产线 | 负债 ≥ 50 kw |

### 🏆 胜利条件

同时满足以下三项即获胜：

1. **资金** ≥ 100 kw（资金 = 现金 + 存款 − 负债）
2. 持有 **3 座五级球场**
3. 获得过 **欧冠冠军**（五级联赛冠军）

此外：3 人及以上局只剩 1 人存活时该玩家获胜；全员破产时最后破产者获胜。

---

### 🎲 回合流程

1. 掷骰子（1–6），前进对应步数
2. 经过起点 +5 kw
3. 处理落点格子效果
4. 结束回合 → 下一玩家

坐牢期间自动跳过回合，每回合减刑 1 轮（共 2 轮）。

### ⚔️ 对战系统

停在他人俱乐部时，若有自己的球队（俱乐部+至少 1 名球员），可选择"挑战"而非支付参观费。

#### 对战规则

| 项目 | 说明 |
|------|------|
| 比赛轮数 | = 主场俱乐部等级（1–5 轮） |
| 每轮流程 | 双方各选 1 名未出场球员 → 掷骰子决定比哪项属性 → 比较属性值 |
| 属性 | 骰子 1=速度 2=射门 3=传球 4=盘带 5=防守 6=身体 |
| 主场优势 | 主队在该属性上 +1（门将不享受） |
| 门将规则 | 门将无六维属性，统一比较 OVR；门将 vs 普通球员：门将用 OVR、普通球员用骰子属性 |
| 胜负 | 属性值高者赢得该轮，平局双方均不得分 |
| 训练点 | 每赢一轮 +1 训练点 |
| 历练 | 出场球员对应属性 +1（门将 OVR+1） |

#### 比分与结算

- **常规时间**：N 轮（N=等级），总分高者胜
- **平局**：进入金球决胜⚡，已出场球员可重新使用，先得分者胜，不限轮数
- **挑战失败**：支付双倍参观费给主队
- **挑战成功**：免费参观
- **联赛积分**：常规胜 3 分，金球胜 2 分，金球负 1 分，常规负 0 分

#### 1–2 级球场挑战限制

Lv1/Lv2 球场只能被**同联赛**俱乐部挑战（如英超球场只能由英超俱乐部挑战）。Lv3+ 不受限。

### 🏅 联赛系统

每级球场对应一个独立联赛，所有在该级别打过比赛的俱乐部自动进入积分榜。

| 级别 | 联赛 | 冠军奖金 | 所需场次 |
|------|------|---------|---------|
| Lv1 | 国内联赛 | 2 kw | 2×人数 |
| Lv2 | 国内杯赛 | 3 kw | 2×人数 |
| Lv3 | 欧协联 | 4 kw | 2×人数 |
| Lv4 | 欧联 | 5 kw | 2×人数 |
| Lv5 | 欧冠 | 10 kw | 2×人数 |

- 达到所需场次后自动结算，按积分排名
- 冠军获得奖金+奖杯，**欧冠冠军是胜利条件之一**
- 亚军获得冠军奖金的 50%
- 并列冠军共享奖金和奖杯
- 奖杯记录在俱乐部上（球场失去后奖杯清除）

### ⚽ 球员系统

共有 **71 名球员**，分为 5 个池：

| 池 | 数量 | 获取方式 |
|----|------|---------|
| 街头食物组 | 16 | 街头足球·食物格（半价） |
| 街头动物组 | 16 | 街头足球·动物格（半价） |
| 转会窗 | 16 | 转会窗竞价（起拍半价） |
| 驻守球员 | 23 | 俱乐部升至 Lv3 自动获得 |
| 青训池 | 15 | 青训学院盲盒（5 kw） |

#### 球员属性

每名球员有六维属性：**速度、射门、传球、盘带、防守、身体**，以及综合评分 **OVR**。门将无六维属性，仅有 OVR。

- 有效 OVR = 基础 OVR + ⌊六维成长总和/6⌋（门将：基础 OVR + 成长）
- 成长上限：有效 OVR ≤ 100
- 球员分配至俱乐部后占用容量（容量 = 俱乐部等级）

#### 转会窗（第 17 格）

1. 系统随机展示一名转会池球员，起拍价 = 身价/2
2. 所有存活玩家从当前玩家开始轮流竞价
3. 每次可加价 **+0.5 / +1 / +2 kw**，也可跳过
4. 仅剩一人未跳过且有出价→成交；全员跳过→流拍
5. 也可选择**出售**自己的非驻守球员：起拍价 = 身价/4，无人竞拍→系统回收
6. 竞得后需分配到有空位的俱乐部

#### 街头足球（第 13 / 28 格）

- 随机从对应池抽一名球员，**购买价 = 身价/2**
- 池空时：所有自己的球员随机属性+1（全门将则 OVR+1）

#### 青训学院（第 39 格）

- 花费 **5 kw** 从青训池随机抽取一名球员（盲盒）
- 池空后无法再抽

#### 训练营（第 29 格）

- 消耗 **1 训练点**，训练一名球员
- 普通球员：六维各 +1（有效 OVR +1）
- 门将：OVR +1
- 训练点来源：对战每赢一轮 +1 点

#### 球员释放与降级溢出

- 可随时释放球员回对应球员池（驻守球员不可释放）
- 球场降级导致球员超出容量时，需逐个处理：转会至其他有空位的俱乐部（免费）或释放回市场
- 球场失去（漏水降级至 Lv1 以下）时：驻守球员直接消失，普通球员需转移或释放

### 🎲 随机事件

踩中随机事件格时从以下事件中随机触发：

| 事件 | 效果 |
|------|------|
| 🎫 刮出彩票 | 50% 概率 1 kw / 50% 概率 2 kw |
| 🛢️ 挖出石油 | +10 kw |
| 📰 特大丑闻 | 现金折半 |
| 💧 球场漏水 | 随机一座自己的球场降 1 级；Lv1 则直接失去 |
| 🎆 燃放焰火 | 罚款 = 持有俱乐部数 × 0.5 kw |
| 🚗 车水马龙 | 收入 = 持有俱乐部数 × 1 kw |
| ⚔️ 巅峰对决 | 选择一名玩家对战，胜者得 5 kw |
| 💰 税务稽查 | 现金最多的玩家缴纳 20% 税款 |

### 🏦 银行系统

| 操作 | 说明 |
|------|------|
| 存款 | 5 或 10 kw，每回合利息 +2% |
| 取款 | 5 或 10 kw |
| 贷款 | 5 或 10 kw，每回合利息 +5%，随时可还 |
| 还款 | 5 / 10 kw 或全部还清 |

- 现金不足支付费用时自动触发**紧急贷款**（补足差额）
- 利息在每轮结束时结算

### ✈️ 机场

- 到达机场后可飞往任意自己的地产（俱乐部或赞助商）
- 没有地产则无事发生

---

### 💾 存档

- 2 个存档槽位，存储在浏览器 localStorage
- 随时保存/读取，存档含当前完整游戏状态
- 刷新或关闭页面不丢进度
